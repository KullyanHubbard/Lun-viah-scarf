# Luneviah Scarf Landing Page

Landing page statis untuk produk scarf dengan HTML, CSS, dan JavaScript vanilla.

## Kondisi Saat Ini

- Proyek berisi 1 halaman utama di `index.html`.
- Styling ada di `style.css`.
- Interaksi halaman ada di `script.js`.
- Aset media disimpan di folder `assets/`.
- Tidak ada framework dan tidak ada proses build.

## Fitur yang Sudah Ada

- Navbar desktop/mobile dengan menu hamburger.
- Smooth scroll untuk navigasi antar section.
- Search produk (desktop dan mobile) untuk memfilter kartu koleksi.
- Animasi reveal saat elemen masuk viewport.
- Efek navbar saat scroll dan parallax ringan pada hero image.
- Layout responsif untuk tablet dan mobile.

## Cara Menjalankan

1. Clone repository ini.
2. Buka `index.html` langsung di browser.

Alternatif: jalankan local server sederhana (contoh `Live Server` di VS Code) lalu akses dari browser.
